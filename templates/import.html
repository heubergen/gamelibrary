Import genres

The Game import requires you to provide the genreId (not the Name) which you can find out in the database in the Genre table.

So it's recommended to import the genres first and then the games.

Make sure you have a file called genres.csv or games.csv in the same directory as the import_csv.py file. The import will append any data that is already in the database (if the genre/game doesn't exist yet), it will not overwrite anything.
